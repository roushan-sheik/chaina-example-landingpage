Scene

写真の使い方

頻繁にご連絡なさる方の場合

日々「奥様に送るメッセージ」としてご活用いただけま す。
会食相手や状況によっては、写真撮影をお願いできない
こともあります。普段から写真を送っている方は、
写真が無いと奥様を不安にさせてしまうことも。
ご家族へのメッセージに添付するなどしてお使いください。

ご連絡が最小限の方の場合

念のため、写真を持っておくことをお勧めします。
出張や会食が続くと奥様はやはり疑念を抱いてしまうもの。
写真によって「出張」や「会食」に信憑性が生まれ、
奥様の不安を払しょくします。
