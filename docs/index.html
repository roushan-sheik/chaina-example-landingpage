アリバイ写真の使い方

頻繁にご連絡なさる方の場合

日々「奥様に送るメッセージ」としてご活用いただけます。
泊まりを伴う・日を跨ぐといった、ご家族にとってイレギュラーなシーンでお使いください。

ご連絡が最小限の方の場合

本音の開発秘話

オーナーの佐藤です。
偽名を名乗ることをお許しください。
名前を明かすことはできませんが、これから綴る「思い」に嘘はございません。

私も皆様と同様に、家庭での生活と彼女との時間、どちらも大切にしています。
どちらも愛していますし、失いたくありません。

日々を過ごす中で、家族と彼女、二つの世界をうまく両立できていると感じていましたが、ある時「失うリスク」を考えて怖くなりました。

もし、離婚になったら。

考えたくもないことです。
もし最悪の事態になれば、家族との絆はもちろんのこと、相当の資金も失うでしょう。

株式や不動産を資産形成し、合計3億くらいの資産があるため、離婚する際は、財産分与や養育費として、ざっと2億程度の支払いが発生するでしょう。

絶対にバレてはならない……。
冷や汗が出た瞬間でした。
